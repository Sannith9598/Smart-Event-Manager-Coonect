# Smart Event Manager Connect

A full-stack event management platform connecting customers with verified event planners.

For complete documentation, setup instructions, and feature details, see the [main README](./frontend_After%20Editing/README.md).

## Quick Start

```bash
# 1. Setup backend
cd "backend_after Editing"
cp .env.example .env   # Fill in your credentials
npm install
npm run dev

# 2. Setup frontend (new terminal)
cd "frontend_After Editing"
npm install
cd "frontend_After Editing"
```

Backend: http://localhost:5000  
Frontend: http://localhost:3000
